# Installation

You need to have virtualenv installed, on debian/ubuntu:

    sudo apt-get install python-dev

Then:

    git clone https://github.com/psycojoker/pokefight
    cd pokefight
    virtualenv ve
    ve/bin/pip install -r requirements.txt
